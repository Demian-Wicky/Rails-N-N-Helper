# Rails N-N Helper

A Ruby program made to help you create N-N database relationships in Ruby On Rails.
<br><br>
<img src="preview/N-N.svg" >

## Setup

Go to the project folder and type:

```ruby
bundle install
```

then:

```ruby
ruby app.rb
```
